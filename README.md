# SMS Number of messages counter + transliteration (cyrillic <-> latin)
## DEMO (w/o saving to db)
https://inspiring-khorana-99886f.netlify.app/

## Project setup
```
npm install
```

### Launch:
```
npm run serve
npm run json-server (to insert data)
```

### Compiles and minifies for production
```
npm run build
```
